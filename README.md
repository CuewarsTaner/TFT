【9.18版本】更新说明 
新神秘箱子机制： 
PVE野怪战斗阶段所掉落的新的神秘箱子将会有3个种类，每个种类可以得到对应的物品。 
一局游戏不同玩家能得到的物品总数和质量，得到了更新，使得游戏更公平了。 
常见箱子：金币、英雄、妮蔻的帮助； 
罕见箱子：装备散件、金币、英雄、妮蔻的帮助； 
金色箱子：金铲、装备大件、金币、英雄、妮蔻的帮助； 
*每名玩家在一局游戏里会得到大致同样数量的神秘箱子； 
*每名玩家在一局游戏里会得到大致同样数量的装备散件；
